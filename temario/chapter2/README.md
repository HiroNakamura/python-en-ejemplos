# Tkinter

En caso de no tener instalado **tkinter** (para Python 3).



# Versión 3

```
$sudo apt-get install python-tk
```


Comprobar instalación.

```
$ python
```


```
> import tkinter
> dir(tkinter)
> quit()
```
